FastAPI를 활용해서 데스크톱 앱을 만들거야.
목록 형태로 Task를 실행하고 API를 통해 데이터를 가져오고 로컬에서 가공한 다음 결과를 view, download, delete할거야.

필요한 요구사항으로는 화면 가운데에 Title과 아래에 Task 목록을 하나씩 Add Task 하는 방식을 사용할거야. 다만 Task는 미리 사전에 정의된 입력 폼을 받아 Run하는 방식이고, 애플 미리알림 추가하는 느낌으로 등록할거야. 목록은 타원안에서 입력값 IP, ID, PW을 입력받고, Command를 드롭박스로 입력받고, Run 버튼을 누르면 Progress Bar가 오른쪽에 생기고 완료되면 View, Download, Delete 버튼을 만들거야. 결과 데이터는 데이터프레임이고 view하면 Task가 펼쳐지면서 아래에 테이블을 표시할거야. 접었다가 펼 수 있어야해. 그리고 Download는 엑셀로 저장할거야. Delete는 해당 Task를 삭제하고 결과값도 삭제할거야. 그리고 추가로 Task를 등록하기 위한 하단의 Add Task와 모든 Task를 삭제하는 Clear 버튼을 만들거야. 

디자인은 모던하고 모든 동작들이 인터렉티브해야해. 심플한 기능으로 심플한 디자인이고 고급스러워보였으면 좋겠어.
